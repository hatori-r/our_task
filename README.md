## 📍Introduction
our taskはタスクを他のユーザーにシェアできるアプリケーションです。  

投稿するタスクには目標達成までの期限を決めることが出来るので他のユーザーに共有して、  
達成する意識を高めることが出来ます。  

またタスクの進捗状況も設定できるので、進捗管理も行いやすくなっております。
